# 2026-07-13 — Bell Commander: UI polish, Calendar overlays, safety fixes

Big day on [[Beyond Bell Commander]]. Everything deployed to the office Pi (`172.16.200.217`) and backed up to bns-tooling (`beta`, through commit `2ed37ba`).

## UI — "Option B" polish + dark mode
- Refined the BNS UI to a flatter, modern **Option B** treatment: soft elevation instead of chunky 3D shadows, hairline borders, tighter radii, lighter body weight, wider 1080p layout.
- **Self-hosted Inter** (embedded woff2) — the CSS asked for Inter but it was never bundled, so it fell back to a fuzzy system font. Now crisp + identical everywhere, offline.
- **Bug fixed:** a PowerShell `Get/Set-Content` round-trip double-encoded every emoji/symbol into mojibake (🔔 → `ðŸ""`). Reversed it losslessly, rebuilt. *(Lesson: never edit UTF-8 files via PS 5.1 Get/Set-Content.)*
- **Dark mode** — the UI is fully token-driven, so dark = a `.bbc.dark` block remapping the CSS custom properties. Sun/moon toggle in the header, persists to localStorage, defaults to OS preference.

## Calendar — layered scheduling (overlays)
Rebuilt from a prototype into a real feature. A day is now **base timetable + additive schedules (overlays) + one-off bells − suppressed bells**, instead of one-template-per-day.
- **Engine** (`compile_day`): merges the layers, resolves clashes by **(time, group)** — most-specific wins (one-off > overlay > base); different-target bells coexist. Tags each bell with its `source`. Tick loop unchanged → all DSP drivers work as-is.
- **API:** `/api/schedule` returns `source` + `conflicts`.
- **UI:** interactive Calendar screen (swap base, apply/remove schedules, add/remove one-offs, skip/restore base bells) with a live server-compiled "What will ring" panel. New **Setup → Schedules** editor to create the overlays.
- **Year / Month views** — toggle; month view shows large per-day cells with the base timetable + schedules + one-off/skip counts.
- **Colour-coding:** optional `#RRGGBB` per timetable/schedule (validated), painted through year cells, month cells and the day sheet. "Today" shows its type colour with a green ring.

## Today timeline overhaul
- Replaced the scrollbar with a **draggable, auto-scrolling tape** — hides the scrollbar, auto-centres on *now*, finger-pans (native momentum) and mouse-drags with **velocity-tracked inertia** (flick-to-glide). Spans the day's real bell range so evening bells spread out.
- **Collision-aware labels** — tightly-clustered bells show as pins (label on hover) instead of overlapping into a smear.
- **Pins reflect the source colour code** (base template / schedule); done stays grey, skipped dashed.
- Enlarged the **Now Ringing** pill + icon and moved it below the Rainy Day/Emergency buttons.

## Safety — EVAC/emergency unmute-all
A **muted zone would miss an evacuation** (AHM `evac_suppress` mutes/unmutes all zones, and on release blanket-unmuted, losing operator state).
- Fix: an **emergency alert** now snapshots the mute state and **unmutes every zone** before its tone plays; **All Clear restores** the snapshot.
- Extended to **all emergency alerts** — `(hold or danger-style) and sounds a tone and not release/stop` → covers Onsite/Off-Site EVAC + Lock In + Lock Out. Test / All Clear / STOP unaffected.
- Verified live on the AHM (silent check): mute Gym → EVAC → all unmuted → All Clear → Gym re-muted.
