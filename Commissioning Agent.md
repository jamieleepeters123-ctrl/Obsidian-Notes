![[]]What started as a commissioning tool has become  feature rich it needs its own commissioning tool. 

The commissioning agent will make the setup process effortless. Walking the installer step by step through a new intuitive ui overlay with the ability to skip steps and go back freely. 

The agent will have 2 paths commissioning and end user

### Commissioning (Admin only)
- School name (colors) two school colors will be able to be selected to customise the ui to the school colors.
- timetable
- calander
- default belltone
- zones
- volume
- EVAC
(All other fileds in the settings should be at there default state)
- The Panel should auto detect the server auto fill the ip field suggestions. 
### User
- timetable
- calander 
- change belltone

The button will be on the main page and should be able complete these basic tasks through its own intuitive ui.
